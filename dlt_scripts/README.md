# DLT Scripts

### Breakdown

So the main reason why I used dlt over Airbyte is because my EC2 is only a t3a.large. I needed something that can be flexible because I planned on using different APIs, but also lightweight.
From what I have read, Airbyte can also act as an orchestrator and has many tools at its disposal. I would love to explore Airbyte in my future projects, but dlt will have to do for now.

I plan on orchestrating these scripts via Airflow. Writing dlt_scripts and DAGs will help me hone my Python skills.
